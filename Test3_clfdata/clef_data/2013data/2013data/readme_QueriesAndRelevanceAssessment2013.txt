README for CLEF eHealth 2013 Patient Queries & Relevance Set
============================================================

The query and result set are distributed as part of the CLEFeHealth 2013 Task 3.

The official training query and result set for 2013 Task 3 consists of 50 queries and corresponding result set generated from manual relevance assessment (by medical professionals) on a shallow pool. It follows the same format as 2014 dataset. Details on its content can be found in 

Lorraine Goeuriot, Gareth J. F. Jones, Liadh Kelly, Johannes Leveling, Allan Hanbury, Henning Müller, Sanna Salanterä, Hanna Suominen and Guido Zuccon (2013) - ShARe/CLEF eHealth Evaluation Lab 2013, Task 3: Information Retrieval to Address Patients? Questions when Reading Clinical Reports. In proceedings of CLEF 2013 (online working notes).
